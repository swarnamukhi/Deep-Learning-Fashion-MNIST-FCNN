# Deep-Learning-Fashion-MNIST-FCNN
# Deep Learning - Fashion MNIST Classification using Fully Connected Neural Network (FCNN)

> A comprehensive deep learning project demonstrating image classification using a Fully Connected Neural Network (FCNN) with TensorFlow/Keras. This repository emphasizes the concepts behind every implementation step, making it a valuable resource for learning, interview preparation, and portfolio development.

---

## 📌 Project Overview

Fashion MNIST is a benchmark computer vision dataset consisting of grayscale images of clothing items. In this project, we build a **Fully Connected Neural Network (FCNN)** to classify images into one of **10 fashion categories**.

Unlike repositories that focus only on implementation, this project explains **why** each step is required, **how** it works internally, and **how** to explain the concepts in technical interviews.

---

## 🎯 Learning Objectives

By completing this project, you will understand:

- Model Evaluation

## 📂 Dataset

**Dataset:** Fashion MNIST

| Feature | Value |
|---------|------:|
| Training Images | 60,000 |
| Training Labels | 60,000 |
| Testing Images | 10,000 |
| Testing Labels | 10,000 |
| Image Size | 28 × 28 |
| Color Channel | Grayscale |
| Number of Classes | 10 |
